# Sponsorship

SAT>IP Lab Server is free and open source. If it helped you debug a client, avoid hardware setup, or add SAT>IP coverage to CI, a one-time sponsorship helps keep the project maintained.

Sponsor through [GitHub Sponsors](https://github.com/sponsors/e12media).

Sponsorship is voluntary support. It does not include consulting, private support, priority fixes, paid feature commitments, roadmap influence, or SLA.

## Suggested GitHub Sponsors tiers

GitHub Sponsors tier amounts are configured in USD.

| Amount | Tier | Meaning |
|--------|------|---------|

| `$15` | Saved me an afternoon | You avoided manual setup or a debugging detour. |
| `$30` | CI smoke test unlocked | You added SAT>IP coverage to CI or caught regressions earlier. |
