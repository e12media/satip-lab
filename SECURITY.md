# Security Policy

## Supported Versions

Security fixes target the latest released version of SAT>IP Lab Server.

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories:

https://github.com/e12media/satip-lab/security/advisories/new

Do not open a public issue for suspected vulnerabilities. Include the affected version, reproduction steps, and any relevant logs or traces that do not contain secrets.

SAT>IP Lab Server is intended for local development and CI environments. It does not provide authentication, HTTPS, or production hardening.
